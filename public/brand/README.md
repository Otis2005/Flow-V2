# TenderFlow brand assets

Canonical logo files. Don't edit these inline — see `docs/BRAND.md` for the rules, and update the SVGs here if the brand evolves.

| File | When to use |
|---|---|
| `logo.svg` | Default primary lockup. Light backgrounds (cream, paper, white). |
| `logo-on-dark.svg` | Primary lockup, dark backgrounds (navy bands, photos, video). |
| `mark.svg` | Three-bar mark only. Favicons, profile pics, contexts that already say "TenderFlow" elsewhere. |
| `mark-on-dark.svg` | Mark only, dark-background variant. |
| `wordmark.svg` | Italic wordmark only, no bars. Inline use, signatures. |
| `wordmark-on-dark.svg` | Wordmark only, dark backgrounds. |

## Pairing

Always pair light-bg variant with light surfaces, dark-bg variant with dark surfaces. The tall bar / wordmark text colour swaps to paper-cream on dark to maintain contrast.

## Colours encoded in these files

- `#0A2240` Navy (primary brand)
- `#C7841A` Amber (mark bar #1, short)
- `#00A98F` Teal (mark bar #2, medium)
- `#FFFDF7` Paper (mark bar #3 on dark, wordmark text on dark)
- `#E7D49E` Gold-soft (italic "Flow" on dark)

## Exporting to PNG

For social media uploads or partner decks that need PNG/JPG:

1. Open the SVG in a browser.
2. Right-click → Save as image.
3. Or use https://cloudconvert.com/svg-to-png — free, no signup, accurate.

Recommended PNG export sizes:
- 256×256 (square, mark only)
- 1024×1024 (square, mark only — Android home screen)
- 1200×400 (rectangular, full lockup — LinkedIn header)
- 1500×500 (rectangular, full lockup — Twitter header)
